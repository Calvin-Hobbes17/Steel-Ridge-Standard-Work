# Standard Appearances

We have standard colors to be used for different materials. They are listed in the table below. You don't *need* to do it, but it makes it look a lot better. You'll thank yourself for it later.

| Material       | Color (HEX) | Opacity (Alpha) |
| :------------: | :---------: | :-------------: |
| Aluminum       | `E6E6E6`    | `1.0`           |
| Polycarbonate  | `EAEAEA`    | `0.2`           |
| Wood           | `F0DFAD`    | `1.0`           |
| Misc Blue      | `1A1A77`    | `1.0`           |
| Misc Black     | `373737`    | `1.0`           |
| Reference Part | `FFC10E`    | `1.0`           |
